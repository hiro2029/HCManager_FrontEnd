# HelloCManager
HelloC Managerのフロントエンドシステムです。

## 概要
- [HelloC Manager](https://github.com/HIT-matsumotolab/HelloC_API)をAPIとしたフロントエンドシステムです。

## インストール・実行方法

- リポジトリをクローン
```
git clone https://github.com/kirby0113/HCManager_FrontEnd.git
```

- パッケージのインストール
```
npm install
```

- 実行
```
npm run start
```

- 本システムは、大学専用のVPN接続が必要となります。
