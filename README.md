# HelloCManager
HelloC Managerのフロントエンドシステムです。

## 概要
- [HelloC Manager](https://github.com/HIT-matsumotolab/HelloC_API)をAPIとしたフロントエンドシステムです。

## 開発環境・技術選定

### OS・フレームワーク等
- Windows 10
- JavaScript
- React (v17.0.2)

### パッケージ

#### CSS in JS
- styled-components (v5.3.1)

#### デザインライブラリ
- material-ui/core (v4.12.3)
- material-ui/icons (v4.11.2)

#### 問題作成のForm管理
- react-final-form (v6.5.7)

#### 問題文表示用
- query-string (v7.0.1)
- papaparse (v5.3.1)
- html-react-parser (v1.4.5)

#### ルーティング管理
- react-router (v5.2.1)

#### ページネーション
- react-pagination (v5.2.1)

## インストール・実行方法

- リポジトリをクローン
```
npm install
```

- ページを表示
```
npm run start
```
